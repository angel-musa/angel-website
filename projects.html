Fortuna

Fortuna is a stock analysis and forecasting app built using Streamlit to help investors make informed decisions. The app offers a comprehensive stock overview, financial metrics, and the latest news for selected stocks. Fortuna integrates machine learning models to forecast stock prices, providing users with interactive visualizations. Additionally, it features a sentiment analysis tool that automatically updates every hour, assessing public sentiment on social media about a given stock. Fortuna also includes a personalized watchlist for users to track their favorite stocks in one place.
